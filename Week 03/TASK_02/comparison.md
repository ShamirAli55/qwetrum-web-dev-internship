# Framework vs Plain CSS Comparison

Tailwind was so much faster for making the layout look good. Instead of jumping back and forth to a CSS file, I just added classes like `grid` and `md:grid-cols-2` directly in the HTML. It was way easier to make it work on my phone too because the responsive classes are built right in.

The hard part was the HTML getting really messy with tons of long classes everywhere. Sometimes I didn't know which class was doing what. Plain CSS is easier if I want to keep my HTML clean, but for a fast landing page, Tailwind wins.
